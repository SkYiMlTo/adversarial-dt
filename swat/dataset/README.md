# SWaT Dataset

This directory contains the SWaT (Secure Water Treatment) dataset used for the offline evaluation (S2 protocol).

## How to get the dataset
If you are cloning this repository from GitHub, the dataset files are not included due to their large size.
You need to download them and place them in this folder.

You can download the dataset from Kaggle:
[SWaT Dataset on Kaggle](https://www.kaggle.com/datasets/vishala28/swat-dataset-secure-water-treatment-system?resource=download)

Alternatively, you can request the official dataset from iTrust:
[iTrust SUTD Datasets](https://itrust.sutd.edu.sg/itrust-labs_datasets/)

## Expected Files
Please ensure the following files are present in this directory:
- `normal.csv` (or `SWaT_Dataset_Normal_v1.csv`)
- `attack.csv` (or `SWaT_Dataset_Attack_v0.csv`)
